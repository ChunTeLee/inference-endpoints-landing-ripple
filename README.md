# Inference Endpoints — ripple / orbit version

Static replica of the Inference Endpoints landing page with the orbit-style
engines panel (rotating engine logos around a center IE chevron, surrounded
by concentric "ripple" rings that fade outward) and a built-in light/dark
toggle in the header.

## Live preview

**https://chuntelee.github.io/inference-endpoints-landing-ripple/**

The header has a sun / moon button that toggles theme. Preference is
persisted to `localStorage` and applied before paint to avoid a light
flash on dark-mode reloads.

- **Light**: Features + Engines + Pricing
- **Dark**: Engines only (Features + Pricing hidden via CSS)

## What's here

- `index.html` — the unified page with toggle. Generated by
  `build_combined.py`.
- `_light_source.html` — pristine light copy extracted from commit
  `325bd67` of the parent replica repo (last orbit version before the
  cube-pattern merge). The build script reads from this so re-running
  is idempotent.
- `build_combined.py` — re-runnable transformer that takes
  `_light_source.html` and produces `index.html`: tags Features
  section and Pricing row with `data-light-only`, injects dark mode
  CSS overrides, adds the toggle button to the header, and adds a
  pre-paint init script + click handler.
- `assets/` — mirrored images, favicons, and engine logos.

## Dark-mode design

Palette is taken from huggingface.co/pricing in dark mode:

- Body background: `#0B0F19`
- Card surface / orbit plates: `#1F2937` (gray-800)
- Headings: `#F9FAFB` (gray-50)
- Body text: `#94A3B8` (slate-400)
- Borders: `rgba(255,255,255,0.08)`

Engine logo plates and ripple rings are darkened via CSS attribute selectors
(`circle[fill="white"]`) — the SVG itself is unchanged.
